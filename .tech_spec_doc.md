## 1. Gambaran Umum

**Aplikasi Web RSI Siti Hajar Mataram** adalah sistem layanan digital rumah sakit berbasis web yang mengadopsi konsep dari *Siloam Hospitals* namun dengan tampilan, animasi dan teknologi modern.
Aplikasi ini bertujuan untuk memberikan kemudahan bagi pasien dalam mencari dokter, melihat layanan kesehatan, melakukan janji temu online, serta mengakses informasi umum rumah sakit.

Fokus utama aplikasi:

* Tampilan modern dan menggunakan warna khas rsi siti hajar emerlad-500 dan responsif (Next.js + shadcn/ui)
* Keamanan autentikasi (Clerk Auth)
* Data real-time (Supabase)
* Frame Motion (Animasi)
* Interaksi cerdas dengan bantuan AI (AI SDK)
* Multi bahasa (indonesia dan inggris)

## ğŸ”’ 5. Kebijakan Keamanan (Row Level Security / RLS)

RLS diaktifkan untuk semua tabel yang mengandung `user_id`.

### Contoh untuk tabel `appointments`

```sql
ALTER TABLE public.appointments ENABLE ROW LEVEL SECURITY;

CREATE POLICY "User dapat melihat janji temu mereka sendiri"
ON public.appointments
FOR SELECT
TO authenticated
USING ((SELECT auth.jwt()->>'sub') = (user_id)::text);

CREATE POLICY "User hanya dapat menambahkan janji temu mereka sendiri"
ON public.appointments
AS PERMISSIVE
FOR INSERT
TO authenticated
WITH CHECK ((SELECT auth.jwt()->>'sub') = (user_id)::text);
```

### Contoh untuk tabel `chat_sessions`

```sql
ALTER TABLE public.chat_sessions ENABLE ROW LEVEL SECURITY;

CREATE POLICY "User dapat mengakses chat mereka sendiri"
ON public.chat_sessions
FOR ALL
TO authenticated
USING ((SELECT auth.jwt()->>'sub') = (user_id)::text);
```
